# 💼 Portfólio Leonardo Santos

Um site de portfólio profissional e moderno para desenvolvedor, com design dark/tech, animações suaves e totalmente responsivo.

## 🎯 Visão Geral

Este é um portfólio pessoal desenvolvido para Leonardo Santos, apresentando suas habilidades, projetos e informações de contato de forma visualmente atraente e profissional.

## 📱 Responsividade

O site é totalmente responsivo e otimizado para:

- 📱 **Mobile** (< 480px): Layout em coluna única, menu hambúrguer
- 📱 **Tablet** (481px - 768px): Layout adaptado com grid flexível
- 💻 **Desktop** (769px - 968px): Layout em duas colunas
- 🖥️ **Large Desktop** (> 968px): Layout completo com todas as funcionalidades

### Breakpoints Implementados
- `480px`: Ajustes para telas muito pequenas
- `768px`: Mudança para menu hambúrguer
- `968px`: Transição para layout desktop completo

## 📊 Performance

- CSS e JS minificados recomendados para produção
- Fontes otimizadas com `display=swap`
- Animações otimizadas com `will-change` e `transform`
- Lazy loading implementável para imagens
- Debounce aplicado em eventos de scroll

## 🔗 URIs e Navegação

### Rotas Principais
- `/` ou `/index.html` - Página principal
- `#home` - Seção Hero
- `#about` - Seção Sobre
- `#skills` - Seção Habilidades
- `#projects` - Seção Projetos
- `#contact` - Seção Contato

### Navegação
- A navegação é feita via anchor links (#)
- Scroll suave implementado entre seções
- Indicador visual da seção ativa na barra de navegação
